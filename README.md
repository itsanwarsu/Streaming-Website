🎬 Movie Streaming Website

A simple movie streaming website built with HTML, CSS, and JavaScript, featuring movie search, responsive navigation, and LocalStorage-based data management.

✨ Features

- 📱 Responsive navbar with hamburger menu
- 🔍 Search movies by:
  - Title
  - Genre
  - Year
- ⭐ Display movie ratings
- 🎥 Dedicated watch/detail page for each movie
- 💾 Store movie data using LocalStorage
- ⚡ Live search with debounce for better performance

🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

🚀 Installation

1. Clone this repository:

git clone https://github.com/itsanwarsu/Streaming-Website.git

2. Navigate into the project directory:

cd movie-streaming

3. Open "index.html" in your browser.

Or use Live Server in VS Code for a better development experience.

⚙️ How It Works

- Movie data is stored in LocalStorage
- Users can search for movies using keywords
- The system filters movies by title, genre, or release year
- Clicking a movie opens its watch/detail page
- The selected movie ID is temporarily saved in LocalStorage
